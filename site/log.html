Журнал проекта

Этапы разработки, отчёты и события проекта

Демонстрация 3D-аватара

Мы внедрили анимированного 3D-аватара, который отображает жесты из словаря. Это альтернатива видеозаписям с жестами, обеспечивающая универсальность.

Морфология жестов

Была реализована привязка жестов к словоформам. Каждое слово может иметь несколько вариантов визуализации в зависимости от контекста.

Словарь с фильтрацией

Интерфейс жестово-русского словаря обновлён: добавлены фильтры по категории, частям речи и языкам. Пользователь может быстро находить нужные записи.

Разработка серверной платформы для аттестации

В рамках проекта была спроектирована и запущена аттестационная платформа на языке Go с использованием фреймворка Fiber и базы данных PostgreSQL.

  • Поддержка JWT-авторизации с обновлением токенов
  • Обработка заявлений, прикрепление сканов документов, проверка валидности данных
  • Веб-сокеты (WebSocket) для синхронного взаимодействия на экзамене: председатель, экзаменаторы, студенты
  • Автоматическое создание и редактирование экзаменов по JestID
  • Генерация ODT-документов с подстановкой сканов и данных
