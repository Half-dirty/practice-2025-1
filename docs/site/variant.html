Описание Resume Creator

Resume Creator — это отдельный проект вариативной части практики, представляющий собой конструктор резюме. Он позволяет быстро сформировать структурированное резюме на основе введённых пользователем данных. Основная цель приложения — упростить процесс создания резюме для тех, у кого нет навыков верстки или дизайна документов. Пользователь вводит личную информацию, сведения об образовании, опыте работы и навыках, а система автоматически формирует оформленное резюме.

Основные возможности

  • Простой веб-интерфейс для ввода информации о себе.
  • Разделы для личных данных, образования, опыта работы и навыков.
  • Автоматическое форматирование введённых данных по шаблону резюме.
  • Возможность скачать готовое резюме в формате PDF.

Технологии

Backend: Python (Flask) для реализации серверной части и логики приложения.

Frontend: HTML, CSS, JavaScript для создания интерфейса ввода данных.

Дополнительно: Библиотека для генерации итогового PDF-файла резюме.
