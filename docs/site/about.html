О проекте

Современные технологии для жестовой коммуникации и автоматизации

Описание

Проект "Платформа жестовой коммуникации" — это комплексная система для изучения, хранения и взаимодействия с русским жестовым языком (РЖЯ). Основной задачей является разработка электронного словаря и связанных с ним компонентов — таких как платформа аттестации переводчиков, 3D-анимации жестов и система морфологической разметки.

Цели и задачи

  • Создание базы данных жестов и их лингвистических характеристик
  • Разработка словаря с интерфейсом на русском и жестовом языках
  • Интеграция анимаций 3D-аватара для отображения жестов
  • Создание платформы аттестации переводчиков РЖЯ (сайт, экзамены, документы)
  • Переход на новый технологический стек: Go, React, PostgreSQL

Технологии

Frontend: React.js, SCSS, Vite.js

Backend: Go, Fiber, PostgreSQL

Дополнительно: WebSocket, JWT, CI/CD, Docker, Blender (для жестов)

Связь с ЦНИИ РЖЯ

Проект реализуется при участии Центрального научно-исследовательского института русского жестового языка. Харламенков А. Е., директор ЦНИИ РЖЯ, является научным руководителем и экспертом проекта.

Дополнительные модули

  • Морфологическая разметка — связывает словоформы с жестами
  • 3D-анимация — эксперимент с заменой видеожестов на анимированного аватара
  • Платформа аттестации — сервер на Go, подача заявлений, WebSocket-сессии
