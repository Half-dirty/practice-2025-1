Руководство по запуску

Пошаговая инструкция по установке и использованию проекта Resume Creator

Установка и запуск

  1. Установите Python (если Python ещё не установлен на вашем компьютере).
  2. Скачайте или клонируйте репозиторий с исходным кодом проекта Resume Creator.
  3. Откройте консоль в папке с проектом и выполните команды по установке необходимых библиотек (включая Flask и pdfkit).
  4. Дополнительно установите wkhtmltopdf — внешнюю утилиту для генерации PDF. Она требуется библиотеке pdfkit. Скачать можно с официального сайта. После установки убедитесь, что путь к её исполняемому файлу прописан в коде (в файле main.py, параметр configuration = pdfkit.configuration(...)).
  5. Запустите приложение командой python main.py. Сервер будет доступен по адресу http://localhost:5000.

Создание резюме

Откройте в браузере страницу http://localhost:5000. В открывшемся веб-интерфейсе заполните все необходимые поля резюме: личные данные, образование, опыт работы, навыки и т.д. После этого нажмите кнопку создания резюме. Приложение сформирует PDF-файл с вашим резюме, готовый к сохранению или печати.
