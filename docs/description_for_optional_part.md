## –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ –∏ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã

–û—Å–Ω–æ–≤–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Å–∏—Å—Ç–µ–º—ã:

- **–í–µ–±-—Å–µ—Ä–≤–µ—Ä (Back-end)** ‚Äì –æ—Ç–≤–µ—á–∞–µ—Ç –∑–∞ –ø—Ä–∏—ë–º –∏ –æ–±—Ä–∞–±–æ—Ç–∫—É HTTP-–∑–∞–ø—Ä–æ—Å–æ–≤. –ü–µ—Ä–≤–æ–Ω–∞—á–∞–ª—å–Ω–æ –±—ã–ª —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω –ø—Ä–æ—Å—Ç–æ–π HTTP-—Å–µ—Ä–≤–µ—Ä –Ω–∞ —Å–æ–∫–µ—Ç–∞—Ö –ø–æ –ø—Ä–∏–º–µ—Ä—É –∏–∑ —Å—Ç–∞—Ç—å–∏ (–æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—â–∏–π –±–∞–∑–æ–≤—ã–µ GET-–∑–∞–ø—Ä–æ—Å—ã –∏ –æ—Ç–¥–∞—é—â–∏–π —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏–µ —Å—Ç—Ä–∞–Ω–∏—Ü—ã). –û–¥–Ω–∞–∫–æ –¥–ª—è –ø–æ–¥–¥–µ—Ä–∂–∫–∏ —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏ (—Ñ–æ—Ä–º—ã, —à–∞–±–ª–æ–Ω—ã, PDF) —Å–µ—Ä–≤–µ—Ä–Ω–∞—è —á–∞—Å—Ç—å –±—ã–ª–∞ –ø–µ—Ä–µ–Ω–µ—Å–µ–Ω–∞ –Ω–∞ Flask. Flask –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –º–∞—Ä—à—Ä—É—Ç–∏–∑–∞—Ü–∏—é URL (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≥–ª–∞–≤–Ω–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞, —Å—Ç—Ä–∞–Ω–∏—Ü–∞ —Å —Å–æ–≤–µ—Ç–∞–º–∏ –ø–æ –∫–∞—Ä—å–µ—Ä–µ, –≥–µ–Ω–µ—Ä–∞—Ü–∏—è —Ä–µ–∑—é–º–µ), –æ–±—Ä–∞–±–æ—Ç–∫—É –º–µ—Ç–æ–¥–æ–≤ GET/POST, —Ä–∞–±–æ—Ç—É —Å —à–∞–±–ª–æ–Ω–∞–º–∏ –∏ —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏–º–∏ —Ñ–∞–π–ª–∞–º–∏. –°–µ—Ä–≤–µ—Ä —Å–æ–¥–µ—Ä–∂–∏—Ç –æ—Å–Ω–æ–≤–Ω—ã–µ –º–∞—Ä—à—Ä—É—Ç—ã: –≥–ª–∞–≤–Ω–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞ ‚Äú/‚Äù, —Å—Ç—Ä–∞–Ω–∏—Ü–∞ ‚Äú/career‚Äù (—Å–æ–≤–µ—Ç—ã –ø–æ –∫–∞—Ä—å–µ—Ä–µ), POST-–º–µ—Ç–æ–¥ ‚Äú/generate‚Äù –¥–ª—è –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ —Ä–µ–∑—é–º–µ, —Å—Ç—Ä–∞–Ω–∏—Ü—É ‚Äú/success‚Äù –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è —Å—Å—ã–ª–∫–∏ –Ω–∞ –≥–æ—Ç–æ–≤–æ–µ —Ä–µ–∑—é–º–µ, –∏ –º–∞—Ä—à—Ä—É—Ç –≤—ã–¥–∞—á–∏ —Ñ–∞–π–ª–æ–≤ ‚Äú/resume/<–∏–º—è —Ñ–∞–π–ª–∞>‚Äù –¥–ª—è —Å–∫–∞—á–∏–≤–∞–Ω–∏—è PDF.
- **–ö–ª–∏–µ–Ω—Ç—Å–∫–∏–π –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å (Front-end)** ‚Äì –Ω–∞–±–æ—Ä –≤–µ–±-—Å—Ç—Ä–∞–Ω–∏—Ü, —Å –∫–æ—Ç–æ—Ä—ã–º–∏ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É–µ—Ç –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å —á–µ—Ä–µ–∑ –±—Ä–∞—É–∑–µ—Ä. –ì–ª–∞–≤–Ω–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞ —Å–æ–¥–µ—Ä–∂–∏—Ç HTML-**—Ñ–æ—Ä–º—É** –¥–ª—è –≤–≤–æ–¥–∞ –ø–µ—Ä—Å–æ–Ω–∞–ª—å–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö (–§–ò–û, –¥–æ–ª–∂–Ω–æ—Å—Ç—å, –æ–ø—ã—Ç, –Ω–∞–≤—ã–∫–∏, –æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ, –∫–æ–Ω—Ç–∞–∫—Ç–Ω—ã–π email). –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–∞ —Å—Ç—Ä–∞–Ω–∏—Ü–∞ ‚Äú–ö–∞—Ä—å–µ—Ä–Ω—ã–π —Å—Ç–∞—Ä—Ç‚Äù —Å –ø–æ–ª–µ–∑–Ω–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–µ–π –∏ —Å–æ–≤–µ—Ç–∞–º–∏ –¥–ª—è –º–æ–ª–æ–¥—ã—Ö —Å–ø–µ—Ü–∏–∞–ª–∏—Å—Ç–æ–≤. –í–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è —Å –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ–º –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –¥–∏–Ω–∞–º–∏—á–Ω–æ: —Ñ–æ—Ä–º–∞ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –±–µ–∑ –ø–µ—Ä–µ–∑–∞–≥—Ä—É–∑–∫–∏ —Å—Ç—Ä–∞–Ω–∏—Ü—ã –±–ª–∞–≥–æ–¥–∞—Ä—è AJAX (Fetch API –≤ JavaScript), –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É—é—Ç –≤–∏–∑—É–∞–ª—å–Ω—ã–µ —ç—Ñ—Ñ–µ–∫—Ç—ã –∏ –∞–Ω–∏–º–∞—Ü–∏–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ø–ª–∞–≤–Ω–æ–µ –ø–æ—è–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å—Ç—Ä–∞–Ω–∏—Ü—ã, –ø–æ–¥—Å–≤–µ—Ç–∫–∞ –∞–∫—Ç–∏–≤–Ω–æ–≥–æ –ø—É–Ω–∫—Ç–∞ –º–µ–Ω—é, –Ω–µ–±–æ–ª—å—à–∞—è **–º–∏–Ω–∏-–∏–≥—Ä–∞** –Ω–∞ —Å—Ç—Ä–∞–Ω–∏—Ü–µ —Å–æ–≤–µ—Ç–æ–≤). –í—Å—ë —ç—Ç–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–æ —Å—Ä–µ–¥—Å—Ç–≤–∞–º–∏ HTML5/CSS3 –∏ JavaScript –Ω–∞ —Å—Ç–æ—Ä–æ–Ω–µ –∫–ª–∏–µ–Ω—Ç–∞.
- **–®–∞–±–ª–æ–Ω—ã –∏ —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏–µ —Ñ–∞–π–ª—ã** ‚Äì –ø—Ä–æ–µ–∫—Ç –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —à–∞–±–ª–æ–Ω—ã HTML (–¥–≤–∏–∂–æ–∫ **Jinja2**, –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–π –≤–æ Flask) –¥–ª—è –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ —Å—Ç—Ä–∞–Ω–∏—Ü. –ù–∞–ø—Ä–∏–º–µ—Ä, `index.html` ‚Äì —à–∞–±–ª–æ–Ω –≥–ª–∞–≤–Ω–æ–π —Å—Ç—Ä–∞–Ω–∏—Ü—ã —Å —Ñ–æ—Ä–º–æ–π, `career.html` ‚Äì —Å—Ç—Ä–∞–Ω–∏—Ü–∞ —Å —Å–æ–≤–µ—Ç–∞–º–∏, `success.html` ‚Äì —Å—Ç—Ä–∞–Ω–∏—Ü–∞ —É—Å–ø–µ—à–Ω–æ–≥–æ —Å–æ–∑–¥–∞–Ω–∏—è —Ä–µ–∑—é–º–µ. –û—Å–æ–±—ã–π —à–∞–±–ª–æ–Ω `resume_template.html` –æ–ø–∏—Å—ã–≤–∞–µ—Ç —Å—Ç—Ä—É–∫—Ç—É—Ä—É PDF-—Ä–µ–∑—é–º–µ: –≤ –Ω—ë–º —Å –ø–æ–º–æ—â—å—é –ø–ª–µ–π—Å—Ö–æ–ª–¥–µ—Ä–æ–≤ (`{{ }}`) –≤—Å—Ç–∞–≤–ª—è—é—Ç—Å—è –¥–∞–Ω–Ω—ã–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è (–∏–º—è, email, –Ω–∞–≤—ã–∫–∏ –∏ –¥—Ä.) –¥–ª—è —Ñ–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–∏—è —Å–æ–¥–µ—Ä–∂–∞–Ω–∏—è —Ä–µ–∑—é–º–µ. –°—Ç–∞—Ç–∏—á–µ—Å–∫–∏–µ —Ä–µ—Å—É—Ä—Å—ã ‚Äì CSS –∏ JS —Ñ–∞–π–ª—ã ‚Äì —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω—ã –≤ –ø–∞–ø–∫–µ `static`. CSS (`style.css`) –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –æ—Ñ–æ—Ä–º–ª–µ–Ω–∏–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ (—Ñ–æ–Ω, —Ü–≤–µ—Ç–∞, —à—Ä–∏—Ñ—Ç—ã, —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏–µ –±–ª–æ–∫–æ–≤), –∞ JavaScript (`script.js`) —Å–æ–¥–µ—Ä–∂–∏—Ç –∫–æ–¥ –æ–±—Ä–∞–±–æ—Ç–∫–∏ —Å–æ–±—ã—Ç–∏–π –Ω–∞ —Å—Ç–æ—Ä–æ–Ω–µ –∫–ª–∏–µ–Ω—Ç–∞.
- **–ú–æ–¥—É–ª—å –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ PDF** ‚Äì –æ—Ç–≤–µ—á–∞–µ—Ç –∑–∞ —Å–æ–∑–¥–∞–Ω–∏–µ PDF-–¥–æ–∫—É–º–µ–Ω—Ç–∞ —Ä–µ–∑—é–º–µ –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ. –í–º–µ—Å—Ç–æ —Ä—É—á–Ω–æ–π –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ PDF ‚Äú—Å –Ω—É–ª—è‚Äù –ø—Ä–æ–µ–∫—Ç –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –ø–æ–¥—Ö–æ–¥ ‚ÄúHTML-—à–∞–±–ª–æ–Ω + –∫–æ–Ω–≤–µ—Ä—Ç–∞—Ü–∏—è –≤ PDF‚Äù. –î–∞–Ω–Ω—ã–µ –∏–∑ —Ñ–æ—Ä–º—ã –≤—Å—Ç–∞–≤–ª—è—é—Ç—Å—è –≤ HTML-—à–∞–±–ª–æ–Ω —Ä–µ–∑—é–º–µ, –∞ –∑–∞—Ç–µ–º –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ **pdfkit** (–æ–±—ë—Ä—Ç–∫–∞ –Ω–∞–¥ —É—Ç–∏–ª–∏—Ç–æ–π wkhtmltopdf) –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –ø–æ–ª—É—á–∏–≤—à–∏–π—Å—è HTML –≤ PDF-—Ñ–∞–π–ª. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–ø–∏—Å—ã–≤–∞—Ç—å —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Ä–µ–∑—é–º–µ —Å –ø–æ–º–æ—â—å—é –ø—Ä–∏–≤—ã—á–Ω—ã—Ö —Å—Ä–µ–¥—Å—Ç–≤ (HTML-—Ä–∞–∑–º–µ—Ç–∫–∞ –∏ CSS-—Å—Ç–∏–ª–∏) –∏ –ø–æ–ª—É—á–∞—Ç—å —Ö–æ—Ä–æ—à–æ –æ—Ñ–æ—Ä–º–ª–µ–Ω–Ω—ã–π PDF-–¥–æ–∫—É–º–µ–Ω—Ç. –ì–µ–Ω–µ—Ä–∏—Ä—É–µ–º—ã–µ PDF-—Ñ–∞–π–ª—ã —Å–æ—Ö—Ä–∞–Ω—è—é—Ç—Å—è –Ω–∞ —Å–µ—Ä–≤–µ—Ä–µ –≤ –¥–∏—Ä–µ–∫—Ç–æ—Ä–∏–∏ `static/resumes` –∏ –≤—ã–¥–∞—é—Ç—Å—è –∫–ª–∏–µ–Ω—Ç—É –ø–æ –∑–∞–ø—Ä–æ—Å—É.

## –õ–æ–≥–∏–∫–∞ —Ä–∞–±–æ—Ç—ã –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è

1. **–û—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ —Ñ–æ—Ä–º—ã.** –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –æ—Ç–∫—Ä—ã–≤–∞–µ—Ç –≥–ª–∞–≤–Ω—É—é —Å—Ç—Ä–∞–Ω–∏—Ü—É –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è –≤ –±—Ä–∞—É–∑–µ—Ä–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `http://localhost:5000/`). –°–µ—Ä–≤–µ—Ä –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç HTML-—Å—Ç—Ä–∞–Ω–∏—Ü—É (`index.html`), –∫–æ—Ç–æ—Ä–∞—è –∑–∞–≥—Ä—É–∂–∞–µ—Ç—Å—è –≤–º–µ—Å—Ç–µ —Å CSS-—Å—Ç–∏–ª—è–º–∏ –∏ JS-—Å–∫—Ä–∏–ø—Ç–æ–º. –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –≤–∏–¥–∏—Ç –ø—Ä–∏–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω—ã–π —ç–∫—Ä–∞–Ω –∏ —Ñ–æ—Ä–º—É —Ä–µ–∑—é–º–µ —Å –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ –ø–æ–ª—è–º–∏ –≤–≤–æ–¥–∞.
